# Краткое описание проекта
**YumYum** — Мобильное приложение для управления домашними продуктами, рецептами и списком покупок.

Проект решает задачу планирования питания:
- сбор продуктовых позиций из чеков (через QR-код),
- хранение и нормализация остатков продуктов,
- проверка, какие блюда можно приготовить из текущих запасов,
- автоматическая генерация списка покупок по выбранным рецептам.

Проект ориентирован на домашних пользователей, которые хотят контролировать запасы продуктов

**Ключевые особенности:**
- мобильный клиент на React Native (Expo),
- backend API на Node.js + Express,
- PostgreSQL как основная база данных,
- логика рецептов, проверки доступности ингредиентов и формирования shopping list.

# Установка
## 1) Клонирование репозитория
```bash
git clone https://vcs.uni-dubna.ru/PED.22/2026-yumyum.git
cd YumYumDev
```

## 2) Установка зависимостей backend
```bash
cd YumYum-server
npm install
```

## 3) Установка зависимостей frontend
```bash
cd ../YumYum-app
npm install
```

## 4) Настройка переменных окружения backend
Создайте файл `.env` в папке `YumYum-server`:
```env
PORT=5000
PG_USER=postgres
PG_HOST=localhost
PG_DATABASE=yumyum
PG_PASSWORD=your_password
PG_PORT=5432
PROVERKA_TOKEN=your_proverkacheka_token
```

## 5) Запуск проекта
Перейти в папку сервера:

```bash
cd YumYum-server
node index.js
```
Перейти в папку приложения:

```bash
cd YumYum-app
npx expo start
```

## 6) Требования к окружению
- Node.js 18+
- npm 9+
- PostgreSQL 14+
- Expo CLI / Expo Go (для запуска мобильного клиента)

---

После запуска доступны основные сценарии:
- сканирование QR-чека и сохранение товаров,
- просмотр/редактирование списка продуктов,
- создание рецептов,
- проверка доступности ингредиентов для рецепта,
- формирование списка покупок.
